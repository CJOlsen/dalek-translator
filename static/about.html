How does it work?

The Dalek Translator is a small Python web application built using the Flask framework. It makes use of the Natural Language Processing Toolkit which is an *awesome* Python library that does analysis of words (to put it lightly, it's really powerful.)

That's cool, can I have more details?

In this application what's happening is the inputted words are being transformed into what are called "synsets" which are sets of synonyms. Then the synset of the word in question is compared to the synsets of verbs that end in "-ate", the most similar synset wins (with a heavy bias towards 'exterminate'). Synsets are compared based on Leacock Chodorow Similarity which is described in the docs in more detail than I can provide (search for lch_similarity).

Why does it take forever to load sometimes?

The Dalek Translator is hosted on a "sleepy" web server, which means that after an hour of inactivity the app goes to sleep to save resources. If it's sleeping when you try to visit there's a wait because it has to wake up and get its bearings. Where am I? What am I? Oh I'm a webserver, better start serving pages...

Sounds like fun, how do I contribute?

If you're familiar with GitHub you can just get crackin'. If I don't accept your changes you can always fork the project and call it "The Better Dalek Translator" then I'll change the name of this one to "The One And Only Original And Forever Best Dalek Translator" and we'll start a war for internet dominance to rival the Time War itself. If you're not familiar with Git and would like to contribute, you essentially fork the project, work on it locally and then submit a 'pull request' to have your changes absorbed into the main project.
